# cobs

[![GoDoc](https://img.shields.io/static/v1?label=godoc&message=reference&color=burlywood)](https://pkg.go.dev/github.com/creachadair/cobs)
[![CI](https://github.com/creachadair/cobs/actions/workflows/go-presubmit.yml/badge.svg?event=push&branch=main)](https://github.com/creachadair/cobs/actions/workflows/go-presubmit.yml)

This repository implements the [COBS][cobs] encoding format in Go. It was
written as an exercise to understand the format, and should be considered to be
of experimental quality. The API should not be considered stable.

[cobs]: https://conferences.sigcomm.org/sigcomm/1997/papers/p062.pdf
